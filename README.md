# iMail

A Chromium-based macOS desktop app that wraps iCloud Mail web, solving two problems with the native Mail.app:

- **Dark mode text color bug** — emails with hardcoded light colors are unreadable in dark mode. Fixed via injected CSS in the reading pane.
- **Hide My Email via SMTP** — the native app requires SMTP configuration that doesn't play nicely with Hide My Email aliases. The web interface sends through Apple's own servers directly, so this is a non-issue.

---

## Project structure

```
imail/
├── main.js              # Electron main process (window, menu, dock badge)
├── preload.js           # Renderer preload (CSS dark mode fix, notifications)
├── assets/
│   ├── icon.png         # Source icon — 1024×1024 PNG (you provide this)
│   └── icon.icns        # Generated by make-icons script
├── scripts/
│   └── make-icons.js    # Converts icon.png → icon.icns using macOS sips/iconutil
└── package.json
```

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Add an icon

Drop a **1024×1024 PNG** at `assets/icon.png`. A good source is [macosicons.com](https://macosicons.com) — search "mail" for community-made icons.

Then generate the `.icns` file:

```bash
npm run make-icons
```

### 3. Run in development

```bash
npm start
```

### 4. Build a distributable `.dmg`

```bash
npm run build
# outputs to dist/iMail-1.0.0-arm64.dmg (Apple Silicon)
#              dist/iMail-1.0.0-x64.dmg  (Intel)
```

Install by opening the `.dmg` and dragging iMail to `/Applications`.

## Features

| Feature | How |
|---|---|
| Always light mode | `nativeTheme.themeSource = 'light'` — immune to system dark mode, consistent email readability |
| Hide My Email | Web interface sends via Apple's servers — no SMTP client needed |
| No traffic light overlap | `WebContentsView` offsets mail content 44px down; traffic lights float in a clean strip above |
| Window drag + double-click zoom | Strip view loaded with `-webkit-app-region:drag` enables standard macOS title bar gestures |
| Unread Dock badge | Polls the Inbox sidebar element every 3 s; shows a `•` dot badge when inbox has unread mail, clears when read |
| Persistent login | `partition: 'persist:imail'` stores cookies in `~/Library/Application Support/imail` |
| External links | Links outside `icloud.com` open in your default browser |

## Session data

Login cookies and site data are stored at:

```
~/Library/Application Support/imail/
```

Deleting this folder logs you out and clears all stored data.
